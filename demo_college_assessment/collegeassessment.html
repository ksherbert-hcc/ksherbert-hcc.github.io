HCC College Competency Reporting Form

Directions

  • Academic directors: consolidate assessment data from all courses taught in your program this semester.
  • On this form, only report data from the final indicator for each course, as this most accurately reflects the highest proficiency level achieved by students.
  • Each course should be categorized according to which competency it assessed (Competency) and to how the course relates to the college curriculum (Data Source). Data from courses with the same competency and data source may be combined into a single row.
  • Alternatively, if you find it easier, you may choose to create a separate row for each course. The "Label" column is provided as a convenience to distinguish rows as you enter data. The database does not require it.
