The Douay-Rheims Bible

The Douay-Rheims translation of the bible is a direct translation of the Latin vulgate. But it is hard to justify calling it a vulgar translation today! Originally published in the sixteenth century, it is conspicuous for its use of Latinized versions of otherwise recognizable names. How many can you find?

Note: The version of the Douay-Rheims available through this API is missing the so-called deuterocanonical books, a collection of Old Testament texts included in the Catholic canon but not in Protestant canon. The original translation certainly does include these books, and I think their omission is an accident of the interface set up for the API. It does appear to include other portions of text which Protestant bibles normally leave out, e.g. chapters 13 and 14 of Daniel.
